=== Описание лицензии

Лицензия, краткими словами, означает, что все ресурсы в этом репозитории, не касающихся `asciidoctor-skins` запривачены для чужих лиц. Использование содержимого этого репозитория без разрешения будет считаться нарушением `ЛИЦЕНЗИИ`

== Содержимое репозитория

Репозиторий содержит исходный `SVG` файл карты и `ADOC` файл механик и описания ВПИ как такового, а также горсть вспомогательных файлов.
